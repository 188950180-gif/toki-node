# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让参与我们的社区成为一个无骚扰的体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性认同和性取向如何，我们都致力于为每个人提供开放、热情、多样化、包容和健康的环境。

我们承诺以有助于建立开放、欢迎、多样化、包容和健康环境的方式行事和互动。

## 我们的标准

有助于为我们的社区创造积极环境的行为示例包括：

* 对他人表现出尊重和友善
* 对不同的观点、经验和想法持开放态度
* 给予并优雅地接受建设性反馈
* 承担责任并向受我们错误影响的人道歉
* 关注对整个社区最有利的事情，而不仅仅是对个人
* 对社区其他成员表现出同理心和友善

不可接受的行为示例包括：

* 使用性化语言或图像，以及任何形式的性关注或性骚扰
* 捣乱、侮辱性或贬损性评论以及人身或政治攻击
* 公开或私下骚扰
* 未经明确许可，以任何方式发布他人的私人信息，例如物理地址或电子邮件地址
* 在专业环境中可能被合理认为不适当的其他行为

## 执行责任

社区领导者负责阐明和执行我们可接受行为的标准，并将针对任何他们认为不适当、威胁性、冒犯性或有害的行为采取适当和公平的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题或其他贡献，并在适当时传达禁止性、临时性或永久性禁令以及其他适度行为的理由。

## 范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中代表社区的任何情况。

代表社区的情况包括使用官方电子邮件地址、通过官方社交媒体帐户发布信息或在线上或线下活动中担任指定代表。

## 执行

可以通过 [在此处插入联系方式] 向负责执行的团队报告辱骂、骚扰或其他不可接受的行为。

所有投诉都将被及时和公平地审查和调查。

所有社区领导者都有义务尊重事件报告者的隐私和安全。

## 执行指南

社区领导者将遵循这些社区影响指南来确定任何行为的后果：

### 1. 纠正

**社区影响**：使用不适当语言或其他被认为不专业或不受社区欢迎的行为。

**后果**：社区领导者发出私下书面警告，明确违规性质并解释为何该行为不适当。可能要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为违反行为准则。

**后果**：警告并说明持续行为的后果。在指定时间内不得与相关人员互动，包括与行为准则执行者的非请求互动。这包括避免在社区空间以及外部渠道（如社交媒体）中的互动。违反这些条款可能导致临时或永久性禁令。

### 3. 临时性禁令

**社区影响**：严重违反社区标准，包括持续的不适当行为。

**后果**：在指定时间内不得与社区内的任何人进行任何形式的互动或公开交流。在此期间不得与相关人员进行任何公开或私下互动。违反这些条款可能导致永久性禁令。

### 4. 永久性禁令

**社区影响**：表现出违反社区标准的模式，包括持续的不适当行为、对个人的骚扰或对某类人的攻击性贬低。

**后果**：永久性禁止在社区内进行任何形式的公开交流。
