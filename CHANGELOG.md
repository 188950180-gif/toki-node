## 版本说明

### 版本号规则

- **主版本号（MAJOR）**：不兼容的 API 更改
- **次版本号（MINOR）**：向后兼容的功能性新增
- **修订号（PATCH）**：向后兼容的问题修正

### 发布周期

- **Patch 版本**：随时发布（Bug 修复）
- **Minor 版本**：每月发布（新功能）
- **Major 版本**：按需发布（重大更新）
